50+ ESP8266 NodeMCU Projects



Each tutorial includes circuit schematics, source code, images and videos. The ESP8266 is an affordable development board that combines control of outputs/inputs and Wi-Fi capabilities. Additionally, the ESP8266 NodeMCU can be programmed using the Arduino IDE programming environment.

It supports the following peripherals: I/O, SPI, I2C, I2S, SPI, UART and ADC.
